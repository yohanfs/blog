#+begin_src 
post 
#+end_src

Daftar post disimpan di blog.org. File blog.org dapat dibuat dengan menjalankan:

#+begin_src 
list
#+end_src

Klik [[./blog.org][BLOG]] untuk menuju ke daftar post.
